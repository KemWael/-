# سَنَد | صندوق تكافل الزواج

موقع تعريفي تفاعلي لمبادرة "سَنَد" — صندوق تكافل مجتمعي لدعم الشباب والفتيات المقبلين على الزواج، عن طريق مساهمات شهرية جماعية.

## 🚀 عرض الموقع مباشرة (GitHub Pages)

بعد ما ترفع المشروع على GitHub:

1. روح لـ **Settings** في الـ repo
2. من القايمة الجانبية اختار **Pages**
3. تحت "Branch" اختار `main` والفولدر `/ (root)`
4. دوس **Save**
5. بعد دقيقة أو اتنين، الموقع هيبقى شغال على رابط زي:
   `https://<اسم-حسابك>.github.io/<اسم-الـ-repo>/`

## 📁 محتويات المشروع

```
sanad-project/
├── index.html          ← الموقع كامل (تصميم + منطق الحاسبات + سجل المساهمين)
├── apps-script-code.gs ← كود Google Apps Script لربط الموقع بجوجل شيت حقيقي
└── README.md            ← هذا الملف
```

## ✨ المميزات

- تصميم زجاجي (liquid glass) بحركات 3D خفيفة
- شرح كامل لآلية عمل الصندوق خطوة بخطوة
- مساران منفصلان: دعم الرجال ودعم السيدات، كل واحد بحاسبته الخاصة
- بوابة دفع تجريبية (إنستاباي / محفظة إلكترونية / تحويل بنكي)
- سجل مساهمين تلقائي بمعيار استحقاق (3 عمليات سداد منتظمة = مستحق للدعم)
- تصدير سجل المساهمين إلى Excel (CSV) بضغطة زر
- عداد تفاعلي مباشر يعرض إجمالي المتبرع به لكل مسار

## 🔗 ربط الموقع بجوجل شيت (اختياري لكن موصى به)

الموقع حاليًا بيخزن بيانات المساهمين محليًا في متصفح كل زائر (لأغراض العرض التجريبي). لعمل سجل مركزي حقيقي يشوفه الجميع:

1. اعمل Google Sheet جديد وسمّي أول صف بالعناوين: `الاسم | وسيلة الدفع | الرقم | المسار | المبلغ | التاريخ`
2. من Extensions افتح **Apps Script**، امسح الكود الموجود، والصق محتوى ملف `apps-script-code.gs`
3. من Deploy → New deployment → اختر **Web app** → Access: **Anyone** → Deploy
4. انسخ رابط الـ Web App اللي هيديهولك
5. في `index.html`، دور على دالة `logDonation` وضيف استدعاء `fetch` لإرسال البيانات لنفس الرابط ده (بالإضافة للتخزين المحلي الموجود)

## ⚠️ ملاحظات مهمة قبل الاستخدام الفعلي

- بوابات الدفع في الموقع **تجريبية وليست حقيقية** — لا يتم تحصيل أي أموال فعليًا. لتفعيل تحصيل حقيقي، لازم حساب مؤسسي موثّق باسم الصندوق (مش حساب شخصي)، بعد استشارة قانونية.
- يُنصح بشدة بتسجيل الكيان رسميًا (جمعية أهلية أو تحت مظلة جمعية معتمدة) قبل جمع أي تبرعات فعلية من الجمهور.
- الشفافية المالية (كشف حساب دوري معلن، قرارات جماعية للجنة) أساسية لاستمرار ثقة الأعضاء.
